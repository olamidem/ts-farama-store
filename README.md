## 📖 Overview

Farama ERP Lite is a lightweight Enterprise Resource Planning (ERP) system that helps businesses manage their day-to-day retail operations from a single platform.

The application combines inventory management, point-of-sale (POS), purchasing, invoicing, stock control, and customer management into one modern, responsive, and scalable solution.

Originally developed as TS-Farama Store, the project has evolved into a more comprehensive ERP solution by adopting React, TypeScript, and Supabase. The focus is on clean architecture, reusable components, secure authentication, and an intuitive user experience while following modern software engineering best practices.

## 🔄 Project Evolution

Farama ERP Lite began as **TS-Farama Store**, a modern POS and inventory management application.

As the project evolved, additional business modules such as purchasing, invoicing, customer management, and reporting were introduced, transforming it into a lightweight ERP solution suitable for small and medium-sized businesses.

The project reflects a continuous focus on clean architecture, scalability, maintainability, and modern software engineering practices.
